# ToDoList

Trabalho da faculdade - sistema de lista de tarefas usando React Native e Node.js

## Como rodar

### Backend

1. Entre na pasta `backend`
2. Instale as dependencias: `npm install`
3. Crie um arquivo `.env` baseado no `.env.example` e coloque a sua connection string do MongoDB Atlas
4. Rode o servidor: `node server.js`

### Frontend

1. Entre na pasta `frontend`
2. Instale as dependencias: `npm install`
3. Abra o arquivo `App.js` e mude a variavel `API_URL` pro IP da sua maquina na rede
4. Rode o expo: `npx expo start`
5. Escaneia o QR code com o Expo Go no celular

## Tecnologias

- Node.js
- Express
- MongoDB Atlas (Mongoose)
- React Native (Expo)
